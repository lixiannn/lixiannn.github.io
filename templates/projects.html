Projects

All these are CS50W/CS50G personal projects that I did in my free time.

Project 1 - Book Review Website

This is a simple book review website. It allows users to search for books according to specific ratings/ reviews, and it also allows users to add their own reviews. In addition, it consists of the basic register and log in functions. In addition, we were supposed to generate an API for the books for anyone to access it. This website makes use of the Python-Flask microweb framework and it uses a static csv file as database. I chose this project because prior to this, I had never built a web application before. As such, it will be a good learning opportunity. From this project, I managed to learn how to use the Flask microweb framework to build a simple web application.

To expand on this project, I can explore using other databases that are non-static. For instance, we can connect this application to an SQL server and makes use of relational database. In addition, anotherarea that we can improve on is the user interface design.

Project 2 - Instant Messenger

This instant messenger is another Python-Flask project that uses quite a substantial bit of Javascript. I chose this project because I thought that this is a pretty interesting yet challenging project idea to implement. From this project, I learnt about Javascript and the idea of session to manage different rooms for different grouops of users to have their conversations concurrently.

To expand on this project, I can explore the idea of memory management as right now, the messages are stored in a dictionary. In addition, the extra feature added which is the deletion of message is not very stable due to synchronisation issues. As such, more can be done to take this project further.

Project 3 - Fifty Bird

This is a CS50G Assignment and the spritesheets are from the assignment. The game is built using LOVE framework. I added the different medal options after a certain amount of score has been obtained. This is my first game and so I learnt about LOVE and the idea of state machines here.

To expand on this project, I can allow the bird to fly at different speed to alter the difficulty level of the game. In addition, I can also introduce more obstacles.

Project 4 - Pong

This is a CS50G Assignment and the spritesheets are from the assignment. I chose this game to do because it is fascinating how the idea of gravity can be used to create a really fun game. For this project, I added a new power up and also introduce more balls into the game.

To expand on this project, I can introduce a variety of obstacles and also more power ups.
